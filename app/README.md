# تطبيق لوحة SaaS LMS

ده تطبيق الواجهة الجاهز للتشغيل.

## المتطلبات

- Node.js 20 أو أحدث
- npm 10 أو أحدث

## التثبيت

```bash
npm install
```

## وضع التطوير

```bash
npm run dev
```

بعد التشغيل افتح الرابط اللي Vite بيعرضه في التيرمنال (غالبًا `http://localhost:5173`).

## طريقة الاختبار

### 1) اختبار نسخة الإنتاج

```bash
npm run build
```

الأمر ده بيتأكد إن TypeScript و Vite قدروا يبنوا نسخة صالحة للنشر.

### 2) اختبار تشغيل نسخة الإنتاج

```bash
npm run preview
```

بعد ما السيرفر يشتغل، افتح الرابط وراجع بسرعة:

- الداشبورد بيظهر بشكل طبيعي
- صفحة الكورسات والفلترة والترتيب شغالين
- صفحة تفاصيل الكورس بتفتح
- التنقل بين الصفحات شغال

### 3) فحص Lint (اختياري)

```bash
npm run lint
```

ملاحظة: المشروع المستورد حاليًا فيه بعض أخطاء lint موجودة مسبقًا في ملفات مولدة تلقائيًا.
